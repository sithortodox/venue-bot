# Roadmap — Venue Bot

**Проект:** Telegram-бот + Mini App для клуба/ивент-площадки
**Заказчик:** Клуб/ивент-площадка
**Стек:** Python (FastAPI + aiogram), PostgreSQL, React + Vite, Docker
**Текущая версия:** Demo v1.0

---

## Demo v1.0 ✅ (текущая — MVP для показа заказчику)

**Цель:** Рабочий прототип для демонстрации заказчику. Бот + Mini App с базовым функционалом.

| Компонент | Статус | Описание |
|-----------|--------|----------|
| Telegram-бот | ✅ Готово | /start, главное меню, навигация |
| Афиша мероприятий | ✅ Готово | Список из API, карточки, детали |
| Бронирование | ✅ Готово | FSM-диалог (меро → имя → телефон → гости) |
| Контакты | ✅ Готово | Telegram, телефон, кнопки связи |
| Mini App | ✅ Готово | 5 экранов: главная, афиша, детали, бронь, профиль |
| FastAPI | ✅ Готово | CRUD: events, bookings, users |
| PostgreSQL | ✅ Готово | 3 модели: users, events, bookings |
| Docker | ✅ Готово | Compose: bot + api + webapp + postgres + redis + nginx |
| Seed-данные | ✅ Готово | 4 тестовых мероприятия |

## v1.1 — Админ-панель (приоритет 🔴)

**Цель:** Управление бронями и мероприятием без выхода из Telegram. Уведомления о заявках.

### Telegram-бот для администратора
- [ ] `/admin` — дашборд (брони за сегодня, ожидающие, выручка)
- [ ] `/approve <id>` — подтвердить бронь → уведомление гостю
- [ ] `/reject <id>` — отклонить бронь → уведомление гостю
- [ ] `/add_event` — добавить мероприятие (FSM: название → дата → зона → цена)
- [ ] `/broadcast` — запустить рассылку (FSM: аудитория → текст → подтверждение)
- [ ] Авто-уведомления в личку админу о новых заявках

### Веб-панель (React)
- [ ] Dashboard: брони за сегодня, выручка, загрузка залов
- [ ] Управление бронированиями: список, фильтры, статусы
- [ ] Список клиентов: контакты, история визитов, баланс депозита
- [ ] CRUD мероприятий: создание, редактирование, удаление
- [ ] Настройки: контакты, зоны, цены

## v1.2 — Билеты и депозиты (приоритет 🔴)

**Цель:** Монетизация. Продажа билетов, система предоплаты через депозиты.

### Билеты
- [ ] Каталог билетов по мероприятиям (Standard / VIP / Early Bird)
- [ ] Покупка через бота и Mini App
- [ ] Генерация QR-кода для каждого билета
- [ ] Проверка билета на входе (сканер QR админом)
- [ ] История покупок в профиле

### Депозиты
- [ ] Пополнение депозита (заявка админу → подтверждение)
- [ ] Автоматическое списание при бронировании
- [ ] История транзакций (пополнение / списание / возврат)
- [ ] Баланс в профиле гостя и в Mini App

## v1.3 — Рассылки и промо (приоритет 🟡)

**Цель:** Удержание аудитории. Повторные касания, акции, персонализация.

### Массовые рассылки
- [ ] Рассылка по всем подписчикам (текст + фото + кнопки)
- [ ] Сегментация: VIP, обычные, неактивные (>30 дней)
- [ ] Планирование рассылок по времени
- [ ] Статистика: доставки, переходы, конверсия

### Триггерные сообщения
- [ ] Напоминание о мероприятии (за 24 часа)
- [ ] Поздравление с днём рождения (если есть дата)
- [ ] Re-engagement (30 дней без визита — "Скучаем!")
- [ ] Подтверждение/отклонение брони → мгновенное уведомление

### Промокоды
- [ ] Создание промокодов (админка)
- [ ] Применение при покупке билета / бронировании
- [ ] Ограничения: срок действия, макс. использований, % или сумма
- [ ] Копирование промокода одним тапом в боте
