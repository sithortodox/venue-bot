# Roadmap — Venue Bot

План развития проекта. Текущая версия: **Demo v1.0** — рабочий бот + Mini App с афишей и бронированием.

---

## Demo v1.0 ✅ (текущая)

- [x] Telegram-бот с главное меню
- [x] Афиша мероприятий (из API)
- [x] Форма бронирования (FSM: мероприятие → имя → телефон → гости)
- [x] Контакты администратора
- [x] Mini App: главная, афиша, детали, бронь, профиль
- [x] FastAPI: CRUD для мероприятий и бронирований
- [x] PostgreSQL: базовая схема (users, events, bookings)
- [x] Docker Compose для запуска
- [x] Seed-скрипт с тестовыми данными

## v1.1 — Админ-панель

- [ ] Telegram-бот для администратора
  - [ ] `/admin` — дашборд (брони за сегодня, ожидающие)

  - [ ] Dashboard со статистикой

  - [ ] CRUD мероприятий
